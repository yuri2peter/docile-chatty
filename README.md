# docile-chatty

Docile Chatty 是一个适用于[CreativeChatGLM](https://github.com/ypwhs/CreativeChatGLM)的客户端，它提供了一个便于用户编辑、存储聊天记录的聊天 UI，致力于充分发挥 CreativeChatGLM 的“续写”能力和测试 ChatGLM。

感谢 CreativeChatGLM 和 ChatGLM 的作者/团队为我们带来这么有趣的项目。
